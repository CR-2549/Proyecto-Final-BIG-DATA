nota; Los links son redireccionables hacía fotos de como debería quedar para que no queden dudas de como está organizado.<br>

----- Procedimiento para la correcta ejecución del código. -----

1.- Descargar ambos archivos, tanto HTML como .js <br>
2.- En mi caso, los tenía juntos en una carpeta llamada "HTML"<br>
3.- Ya estando en la carpera arrastrar ambos archivos hacía visual code y queda de la siguiente manera: https://gyazo.com/d397b8f2957f0793b4343c519ab4bcbb<br>
4.- Una vez estan ambos archivos, para evitar que windows falle a la hora de ejecutar la camara o tener errores de seguridad, instalaeremos una extensión 
    llamada "Live Server" del autor "Ritwick Dey" en Visual Code <br>
    --- Mini tutorial para instalar extensión ---<br>
    1.- link con la información base, guiarse con los colores, rojo, verde y blanco (En ese orden)<br>
    2.- Picarle al boton de extensiones (Rojo)<br>
    3.- Escribir: "Live Server Ritwick Dey" en el recuadro marcado (Verde)<br>
    4.- En mi caso me sale "Unistall" porque ya lo tengo instalado. A ustedes les deberá salir en "Install" (Blanco)<br>
    --- Fin del tutorial ---<br>
5.- Una vez instalada la extensión le damos clic derecho al archivo con terminación HTML y darle clic a la opción "Open with Live Server"
    https://gyazo.com/4af85d49b264ffc2c9c645363e3e8928<br>
6.- Si seguiste los pasos de manera correcta, el programa se ejecutará correctamente: https://gyazo.com/298a92ba8280be10ac26684036329b98<br>

PD: Puede que al ejecutarlo por primera vez te solicite acceso a la camara, solo debemos darle permisos de ejecución. :)
